#  AI Course Recommendation System

An AI-powered Course Recommendation System developed using Python, Pandas, and Scikit-Learn. This project recommends the most relevant courses based on user interests using TF-IDF Vectorization and Cosine Similarity.

---

##  Features

- Course recommendation based on user interests
- Multiple interest support (comma-separated input)
- TF-IDF Vectorization
- Cosine Similarity for intelligent matching
- Recommendation score for each course
- Displays Top 5 recommended courses
- User-friendly console interface
- Error handling for invalid interests

---

## Technologies Used

- Python 3
- Pandas
- Scikit-Learn
- TF-IDF Vectorizer
- Cosine Similarity

##  Installation

Clone the repository

```bash
git clone https://github.com/Tahawaseem13/Decodes-Lab-P3.git
```

Go to project folder

```bash
cd Decodes-Lab-P3
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python main.py
```
